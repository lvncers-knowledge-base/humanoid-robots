# makers

おっけー、今のヒューマノイドAI（人型ロボット）の**世界でよく名前が出るメーカー＆代表的なロボット**をざっくりまとめるね！
最新の商用化や実証運用まで進んでるのもあるよ

## 海外・最前線で名前が出るメーカー

### **Tesla（テスラ） — Optimus**

* 「Optimus」っていう人型ロボットを開発中。
* 工場や物流での自動化・将来的には家庭支援も目標にしてるやつ。
* 量産や実用の動きが注目されてるよ（2026〜）。([ロボスタ - ロボット･AI情報WEBマガジン][1])

### **Boston Dynamics（ボストン・ダイナミクス） — Atlas**

* 昔から話題の「Atlas」は人間そっくりに歩いたりバランス取ったりできるタイプ。
* 最新は完全電動の商用バージョンが出てきて、2028年頃から米工場で実際に稼働予定。([ウィキペディア][2])

### **Figure AI — Figure 02**

* 実際に工場で動いてる例もある（BMWとの実証）。
* 高い自由度＆AI統合で複雑な作業も可能なタイプ。([webtechtin][3])

### **Agility Robotics — Digit**

* 二足歩行ロボでAmazonなどの倉庫テストが進んでるタイプ。
* 重いものの運搬補助とかに使われつつある。([webtechtin][3])

### **1X Technologies — Neo / Eve**

* 家庭やパーソナル用途を狙ったAIロボット。
* まだ量産前だけど“物理AI＋ヒューマノイド”って形で注目。([AIJACK][4])

## 社会的・接客系や研究系のロボット

### **Engineered Arts — Ameca**

* 人と話したり表情リアクション重視のヒューマノイド。
* 研究機関やイベント用でよくデモされる。([ウィキペディア][5])

### **Hanson Robotics — Han / Sophia**

* 「Sophia」や「Han」みたいな顔表情重視の社交型ロボ。
* 会話や対面コミュニケーションで人気。([ウィキペディア][6])

### **SoftBank Robotics — Pepper**

* ヒューマノイドに近い形で、接客・受付とかで実際に使われてる！
* 感情認識とか会話もできる系。([Aegis Softtech][7])

## 中国勢もめっちゃ増えてる！

中国でも多数の企業がヒューマノイドを出してて、**量産実績も増えてきてる**よ。
Leju Roboticsみたいに教育用途〜実務用途までシリーズ製品を出す動きとかある。([ウィキペディア][8])

※ただし、150社以上が「ヒューマノイド開発」とうたってても、実際に量産できるとこは限られてるって指摘もある。([ロボスタ - ロボット･AI情報WEBマガジン][9])

## 日本の状況

日本は昔からロボット研究は超強いけど、**実用的なヒューマノイドAIの量産・一般販売**って点では欧米・中国に追いつかれてる感じ。
例として昔のホンダのASIMOみたいな先行例はあったけど、今の産業用AIロボの量産レベルとは少し違う。([it.impress.co.jp][10])

## ざっくり分類

| 用途      | 代表例                     | 特徴         |
| ------- | ----------------------- | ---------- |
| 工場/物流   | Atlas（Boston）、Figure 02 | 重作業・自動化向け  |
| 倉庫/運搬   | Digit（Agility）          | 単純反復作業の補助  |
| 社会対話/接客 | Pepper、Ameca、Sophia     | 会話・表情対話重視  |
| 生活/家庭向け | Optimus、Neo             | パーソナル支援目指す |

まとめると、**「工場・物流系の実用化進行中」「接客・社交系のデモ活発」「家庭支援系はまだ試作段階」**って感じ！
この分野、技術進んでてもう結構リアルだよ〜🤖
